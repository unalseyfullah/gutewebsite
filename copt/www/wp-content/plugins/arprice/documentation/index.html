Installation And Activation

You can install plugin either of these two methods.

New install via FTP

  1. Download the plugin from CodeCanyon, and extract it. You should find a directory arprice.
  2. In your WordPress installation, upload arprice to wp-content/plugins/
  3. Activate the "ARPrice" plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin from CodeCanyon
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on choose file and browse the download package from CodeCanyon. When the plugin uploads successfully, click on Activate Plugin.

After activating the plugin, you will be able to see ARPrice menu in admin left menu.

After activating the plugin, you need to activate your license for support and automatic upgrades. You will get Pricing Table Analytics Feature only after activating your license. A license key only be used for one installation of WordPress at a time.

Click on Settings link under "ARPrice" Menu to activate your license. You will see below section there on that page. To activate your license, you need to enter your Name, Email address and purchase code( Unique code obtained while purchased this product).

ARPrice pricing table provides you to export your templates as well as to import previously exported templates.

Export Templates

Here you will have list of all default templates as well as your clone templates. You will be able to select multiple templates and export them. You will get text file after exporting templates which will be used to import them again.

Import Templates

Here you will be able to import your previously exported templates from ARPrice.

You can import only text file exported previously from this plugin only.

Global Settings

If you wish to set css class or width at responsive view globally which apply to all your templates then ARPrice pricing table provides you global settings option.

Global Custom Css

Here you will be able to add global css class according to your requirements which will be applied to all your templates globally.

Responsive Settings

Here you will be able to set maximum width of templates globally at mobile view, tablet view as well as desktop view.

Add Character Sets

ARPrice pricing table plugin provides facility to add character sets with google fonts according to your requirements. Here you will have number of options to choose the character sets you want to add with google fonts.

Track button click of pricing table

This option provides facility to enable or disable analytics of clicked columns.

Load JS & CSS in all pages

You may face some css issues with your theme, ARPrice allows you to load css and js files at all pages where even arprice shortcode not exist.

If you required to enable this facility please check Load JS and CSS
